# Semana Omnistack 11

Minha implementação da Semana Omnistack 11, com algumas alterações do projeto original.

- Frontend em React
- Backend em Typescript com uso do SQLite como banco
- Mobile em React Native com o uso do Expo

## Bibliotecas

### Frontend

- material-ui/core e material-ui/icons
- axios
- react-router-dom
- styled-components

### Backtend

- cors
- express
- knex
- jest
- nodemon
- supertest
- celebrate
- cross-env

### Mobile

- react-navigation
- axios
- expo
- expo-constants
- expo-mail-composer
- intl
